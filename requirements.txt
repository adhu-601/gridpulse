# Streamlit Community Cloud installs from this file. The dashboard reads the

# from the ingestion / transform / orchestration layers, so this is deliberately
# a lean, pinned, known-good set — not the full pipeline toolchain.

# or install the package with its extras:
#   pip install -e ".[orchestration,transform,dashboard,stream,dev]"

# Native stack pinned to a conservative, ABI-compatible combination.
# pandas 3.0 / numpy 2.5 segfaulted DuckDB's fetchdf() on the Linux cloud wheel
# (SIGSEGV in the native pandas-DataFrame conversion), even though the same pins
# ran fine on the Windows dev machine. duckdb 1.5.4 is kept so the committed slim
# warehouse (data/gridpulse_dashboard.duckdb) still opens without a rebuild.
